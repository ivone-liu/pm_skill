# pm_skill

This repository contains the **PRD Writing with Diagrams** skill package.

## Included content

- `SKILL.md`: skill definition and working instructions
- `assets/`: PRD template and example diagrams
- `references/`: writing principles, detail guides, checklist, and prompting examples
- `scripts/`: PRD validation and SVG diagram rendering helpers

## Current package

The current skill is designed to:

- write structured PRDs from scattered input
- add stronger product-detail coverage
- include technical implementation thinking
- ask clarifying questions when the user's information is incomplete or ambiguous
- generate PRD diagrams such as flows, swimlanes, wireframes, and state-related visuals

pm_skill/
