# Example Prompts

## Example 1: complete input
请基于以下需求写一份可评审可落地的 PRD，并补充主流程图、泳道图和技术实现思路。

## Example 2: incomplete input
我想做一个后台功能，给运营审核内容用，但我现在只有一些零散想法。请你先识别缺失的关键信息，适当追问，再输出 PRD。

## Example 3: revision mode
以下是旧流程，请基于新增约束写修订版 PRD，重点标明改动点、影响范围和回滚预案。

## Example 4: MVP mode
我只想先做 MVP，请把 PRD 收敛到最小可上线范围，并明确 scope out。
